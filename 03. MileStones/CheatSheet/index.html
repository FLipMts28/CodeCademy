Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The Set of rows defining the collum headers in a table
<tbody> Table Body The Set of rows containing the actual table data
<tr> Table Row The table row container
<td> Table Data The Data set to fulfill the row
<thfoot> Table Foot The Set of rows defining the footer in a table
